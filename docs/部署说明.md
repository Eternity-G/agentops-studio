# AgentOps Studio 部署说明

## 本地运行

安装依赖：

```bash
python -m pip install -e ".[dev]"
```

启动服务：

```bash
python -m uvicorn app.api.routes:app --reload
```

访问页面：

```text
http://localhost:8000/
```

访问接口文档：

```text
http://localhost:8000/docs
```

## Docker 运行

构建镜像：

```bash
docker build -t agentops-studio .
```

启动容器：

```bash
docker run --rm -p 8000:8000 --env-file .env agentops-studio
```

如果只使用 MockProvider，可以不传 `.env`：

```bash
docker run --rm -p 8000:8000 agentops-studio
```

## DeepSeek 配置

本地 `.env` 示例：

```bash
MODEL_PROVIDER=deepseek
DEFAULT_MODEL=deepseek-v4-flash
DEEPSEEK_BASE_URL=https://api.deepseek.com
DEEPSEEK_API_KEY=你的本地 API key
```

注意：

- `.env` 已被 Git 忽略。
- 不要把真实 API key 写入 README、测试、提交信息或 Docker 镜像。
- CI 和公开部署环境应使用平台提供的环境变量管理功能。
